# Tipografía ERP

## Fuentes recomendadas
- Títulos: `Montserrat` (600/700)
- Cuerpo: `Source Sans 3` (400/500)
- Monoespaciada (código/datos): `JetBrains Mono` (400/500)

## Escala tipográfica
- H1: 36px / 44px
- H2: 28px / 36px
- H3: 22px / 30px
- Body L: 18px / 28px
- Body: 16px / 24px
- Small: 14px / 20px
